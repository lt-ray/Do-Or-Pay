# Do-Or-Pay

## 概要

Do-Or-Pay は，Firebase Authentication と Cloud Firestore を利用したタスク管理Webアプリです．  
ユーザーはログイン後，自分のタスクを追加，表示，更新，削除できます．

## 主な機能

- ユーザー認証
- タスクの追加
- タスク一覧表示
- タスクの更新
- タスクの削除
- ログインユーザーごとのデータ管理

## 使用技術

- Next.js
- React
- TypeScript
- Firebase Authentication
- Cloud Firestore
- Firebase Hosting
